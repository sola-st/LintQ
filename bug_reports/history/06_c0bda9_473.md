# What is happening?
In the file [`circuit_0_5.py`](https://github.com/beaudoco/qiskit-shifting-simulator/blob/5348151514ed2c60619b813b7b0eaad693fd8e59/circuits/circuit_0_5.py) the circuit [`qc_5_close`](https://github.com/beaudoco/qiskit-shifting-simulator/blob/5348151514ed2c60619b813b7b0eaad693fd8e59/circuits/circuit_0_5.py#L34) is defined is defined with 5 qubits but it actually operates only on four of them (`0,1,3,4`), leaving the index `2` untouched and making me question on the usefulness of the two (index: `3,4`) which are used but never measured, nor entangled.
